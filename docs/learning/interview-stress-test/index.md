# Interview Stress Test

本目录用于从 Agent 应用开发工程师面试官视角，对项目关键模块做高压审视。

文档目标不是做情绪化批评，而是回答三个更有价值的问题：

1. 这个模块在项目里到底是怎么实现的
2. 对于旅行垂类 Agent，这样的实现到底哪里不够
3. 如果要修，什么是最佳实践，什么是最小投入高收益方案，什么是大厂务实方案
