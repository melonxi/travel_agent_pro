      核心结论

      如果只看 UI，初始化像是在“打开一个空页面”。 但从 agent 视角看，初始化真正完成的是一件更重要的事： 它把 plan、messages、agent、 system prompt 和 phase 对应的 tools 组到一起， 让第一轮 loop 从一开始就在一个已定义好的语义环境里工作。

      所以初始化不是外围细节，而是 loop 的起跑线。 如果这一步没有把运行现场准备完整，后面的阶段推进、工具调用、约束校验和 SSE 展示都会变成松散的拼接，而不是一个可解释、可联调的 agent 系统。
